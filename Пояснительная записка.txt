В проекте 2 players реализовано следующее:
1)Главное меню
Игры:
2)"Танки"
3)"Курицы"
4)"Жабы"
Главный концепт игр - управление персонажем всего 1 кнопкой, что позволяет играть на одном компьютере вдвоем

1. Главное меню - набор кнопок(class Button) при нажатии открывается нужная игра в этом же окне
Использованные технологии: Pygame

2. "Танки" - Игра на  двоих что-то типо примитивного шутера, основные классы:
	class Bullet - патрон, выпускаемый танком
	class Tank - класс танка, которым управляет игрок, отвечает за все связанное с движением и стрельбой
	Использованные технологии: Pygame, math

3. "Курицы" - Игра на двоих в виде платформера, где при нажатии кнопки "меняется гравитация"
	def main - главная функция запуска игры
	def load_level - загрузка уровня
	def generate_level - генерация уровня
	def load_image - загрузка изображения
	def load_platform - создание платформ
	class Chicken - класс курицы, которой управляет игрок
	class Platforms - класс платформ, по которым движутся игроки
	class Camera - класс камеры
	константы
	
4. "Жабы" - Игра на двоих с жабами, прыгающими по воздуху (flappy bird)
    class Player - класс игрока
    class Background - класс фона отдельно
    class Camera - класс камеры
    class Death - класс объекта, которого нельзя касаться
    class Platform - класс объекта, по которому можно ходить
